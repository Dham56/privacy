Privacy Policy — LinkSheet

Effective Date: March 11, 2026

Overview

LinkSheet is a desktop application developed by CropSettle LLC. This privacy policy explains how LinkSheet handles your data.

The Short Version

LinkSheet does not collect, transmit, or store any of your data. Everything happens on your computer. We never see your files.

Data Collection

LinkSheet does NOT collect, transmit, or share:

  • Personal information
  • File names or file contents
  • Spreadsheet data
  • Usage analytics or telemetry
  • Crash reports
  • IP addresses
  • Location data
  • Any other data of any kind

How LinkSheet Works

LinkSheet runs entirely on your local computer. When you use the application:

  1. You select an Excel file and a folder of documents on your computer
  2. LinkSheet reads the Excel file and scans the folder — locally, on your machine
  3. Hyperlinks are created in your Excel file — locally, on your machine
  4. A backup of your original file is saved — locally, on your machine
  5. A log file and report are saved — locally, on your machine

No data is sent to CropSettle LLC, any third party, or any external server. LinkSheet does not connect to the internet for any reason.

Data Storage

All files created by LinkSheet (backups, logs, reports) are stored on your computer in locations you can see and control. You can delete any of these files at any time.

Third-Party Services

LinkSheet does not use any third-party services, APIs, analytics tools, or tracking software.

Changes to This Policy

If we ever change this policy, we will update the date above. Because LinkSheet collects no data, meaningful changes to this policy are unlikely.
